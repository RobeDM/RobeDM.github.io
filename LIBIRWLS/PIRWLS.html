Support Vector MAchines

Suppose a binary classification problem with a data set $\mathcal{D}$ that contains $n$ training examples in a $p$-dimensional space of the form:

$\mathcal{D}=\{(\mathbf{x}_i,y_i)|\mathbf{x}_i \in \mathbb{R}^p,y_i \in \{-1,1\}\}_{i=1}^{n}$

where $y_i$ indicates the class of the training sample $\mathbf{x}_i$.

The primal optimization problem to solve a SVM when we use soft margin classification \cite{SOFT} to allow samples inside the classification margin and using a kernel functions \cite{VAPNIK} is:

$\min_{\mathbf{w},b} frac{1}{2}\Vert\mathbf{w}\Vert^2+C\sum_{i=1}^n{\xi_i}$
Subject to: $y_i(\sum_{j=1}^n\mathbf{w}_jK(\mathbf{x}_j\mathbf{x}_i)+b)\ge 1-\xi_i,\forall i=1,2,...,n$
$\xi_i \ge 0,\forall i=1,2,...,n$

Where $C$ is a penalty parameter to tradeoff the misclassification error and $K(\mathbf{x}_j\mathbf{x}_i)$ is the kernel function to create non-linear solutions. If the kernel is a radial basis function, also called Gaussian kernel $K(\mathbf{x}_i,\mathbf{x}_j)=exp(-\gamma\Vert \mathbf{x}_i - \mathbf{x}_j \Vert^2)$, the corresponding feature space is a Hilbert space of infinite dimensions. This optimization problem is equivalent to solving an $n$-dimensional convex Quadratic Programming (QP) problem. The solution is non-parametric, the size of the classification function (that is the number of Support Vectors) is not predetermined and is modeled based on the training samples. To keep the complexity under control a semi-parametric models can be adopted.
