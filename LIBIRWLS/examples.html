RUNNING THE ALGORITHMS

After building the code, the executable files appear in the lib folder.
Input datasets

The datasets must be provided in LibSVM format:

  • Training: Labeled dataset
  • Testing: Labeled or unlabeled dataset

Labeled format - Every line is a sample and is composed by the class (+1 or -1) and followed by pairs of feature index feature value:

+1 1:5 7:2 15:6
+1 1:5 7:2 13:6 23:3
-1 2:4 7:3 10:6 23:1

Unlabeled format - Every line is a sample and is composed by pairs of feature index and feature value:

1:5 7:2 15:6
1:5 7:2 13:6 23:3
2:4 7:3 10:6 23:1

Running a parallel semiparametric SVM using the IRWLS procedure:

Command line:

./PSIRWLS-train [options] training_set_file model_file

  • training_set_file: Training set in LibSVM format
  • model_file: File where the classifier will be stored

Options:
  • -k kernel type:
    • 0 = Linear kernel u'v
    • 1 = radial basis function exp(-gamma|u-v|^2) (default 1)
  • -g Gamma: Set gamma in the radial basis kernel function (default 1)
  • -c Cost: Set the SVM Cost (default 1)
  • -s Classifier_size: Size of the classifier (default 50)
  • -t Number_of_Threads: It is the number of threads in the parallel task (default 1)
Example:

/Users/robedm/LIBIRWLS

  • ./bin/PSIRWLS-train -g 0.1 -c 100 -t 4 -s 90 training.txt model.mod

Running a parallel SVM using the IRWLS procedure:

Command line:

./PIRWLS-train [options] training_set_file model_file

  • training_set_file: Training set in LibSVM format
  • model_file: File where the classifier will be stored
Options:
  • -k kernel type:
    • 0 for Linear kernel u'*v
    • 1 for radial basis function exp(-gamma*|u-v|^2) (default 1)
  • -g Gamma: Set gamma in the radial basis kernel function (default 1)
  • -c Cost: Set the SVM Cost (default 1)
  • -w Working_set_size: Size of the Least Squares Problem in every iteration (default 500)
  • -t Number_of_Threads: It is the number of threads in the parallel task (default 1)
  • -e eta: Stop criteria (default 0.001)
Example:

/Users/robedm/LIBIRWLS

  • ./bin/PIRWLS-train -g 0.001 -c 1000 -t 4 training.txt model.mod

Using a trained model to classify a new dataset:

Command line:

./LIBIRWLS-predict [options] dataset_file model_file output_file

  • dataset_file: Dataset set in LibSVM format (labeled or unlabeled)
  • model_file: File that contains a classifier trained with PIRWLS or PSIRWLS
  • output_file: File to save the predictions
Options:
  • -t Number_of_Threads: It is the number of threads in the parallel task (default 1)
  • -s Soft output (default 0):
    • 0 Class prediction (the output is +1 or -1)
    • 1 Soft output: The output after the hard decision that decides the class (useful to use in ensembles with other algorithms).
  • -l Labeled: (default 0)
    • 1 if the dataset is labeled (shows accuracy)
    • 0 if the dataset is unlabeled
Example:

/Users/robedm/LIBIRWLS

  • ./bin/LIBIRWLS-predict -t 4 -l 1 test.txt model.mod output.txt
